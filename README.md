**Vala-Checksum** est une Gui développé en Vala et Gtk3+. L'application permet de calculer et de comparer les hashs des fichiers.

---

#Compilation
	Pour compiler **Vala-Checksum**
	```
		~$ valac --pkg gtk+-3.0 ChecksumGtk.vala
    ```
---

#Hashs supportés
	* MD5
	* SHA1
	* SHA256
	* SHA512
